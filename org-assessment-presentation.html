Phase Flow & Scoring Rubric

Four gated phases,
one cumulative HTML report

Every observation must cite a real artifact — no invented findings. Human approval required before HTML is generated.

Phase 0

Scope

Select domains then sub-domains in a two-step conversation. Present scope summary for confirmation.

✋ User replies "confirmed"
→
Phase 1

Ingest

Read OrgCheck, Code Analyzer, Splunk, and metadata artifacts. Extract concrete signals — names, counts, metric values only.

→
Phase 2

Draft Findings

Write observations to findings-review/[SUB-CODE].md. Offers Revise / Append / Replace if draft exists.

✋ User replies "approved"
→
Phase 3

HTML Report

Re-reads approved markdown. Adds or replaces only new panels — existing sub-domain panels are preserved.
